The conference is broadly organized into:

  • Pre-recorded presentations, available for viewing anytime after 21 July 2020 - please view these before the interactive sessions!
  • RocketChat channels to enable focused discussion around presentations.
  • Live Zoom sessions for Q&A and social events - see schedule for Thursday and Friday. Note that this schedule includes the interactive sessions, and does not include time for watching the pre-recorded presentations.

Types of content:

  • Keynote talks
  • Tutorials
  • Research paper talks
  • Workshop spotlight talks
  • Doctoral symposium talks

Note about Zoom links and passwords:

  • Each session has a dedicated zoom link. Visit the the Thursday and Friday schedule pages to to find the corresponding link to each live Q&A session.
  • Registered participants will receive a single password on each day for the all sessions via email. For example, the password for all sessions Day 1 will be XXXXXX and the password for all sessions on Day 2 will be XXYYXX.
  • Please do not share/tweet/post any passwords, as sessions are likely to be Zoom-bombed if these are leaked.

How to interact and ask questions:

  • Realtime questions and coordination can be done over RocketChat via the website, or you can download the RocketChat client and use server https://chat.chilconference.org.
  • You can ask questions in the Zoom webinar using the Q&A function.
For more information, see the FAQ.
